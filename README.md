# Adaptive Exam Question Paper Generator

This project is a web-based application that automatically generates balanced and optimized exam question papers using classical algorithm design techniques.

## 🚀 Features

- Generates question papers based on:
  - Total marks
  - Number of questions
  - Difficulty distribution (Easy, Medium, Hard)
  - Multiple subjects (Maths, Physics, Computer Science, Chemistry, English)

- Ensures:
  - No duplicate or similar questions
  - Balanced difficulty levels
  - Proper subject diversity
  - Optimal marks utilization

## 🧠 Algorithms Used (As per syllabus)

- Greedy Algorithm → Difficulty balancing  
- Dynamic Programming (0/1 Knapsack) → Marks optimization  
- Backtracking → Constraint satisfaction  
- Merge Sort (Divide & Conquer) → Sorting final output  
- String Matching:
  - KMP → Exact duplicate removal  
  - LCS → Similarity detection  

## ⚙️ Key Highlights

- Fully deterministic (No AI/ML used)
- Procedurally generated dataset (150+ questions)
- Real-time algorithm logs for visualization
- Randomized paper generation (different output each time)
- Export to PDF functionality
- Modern UI with glassmorphism design

## 🖥️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Enter constraints and generate paper  

## 🎯 Objective

To demonstrate how classical algorithm design techniques can be applied to solve a real-world optimization problem like automated exam paper generation.
